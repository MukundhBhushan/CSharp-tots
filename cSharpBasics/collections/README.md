#Non generic
- Not type specific
- Add/Remove elements at run time
- Dynamic in size
- Name space: System.Collections
- types: ArrayList, Hash table, Sorted List, Stack, Queues

#Generic
- type specific
- Add/Remove elements at run time
- Dynamic in size
- Name space: System.Collection.Generic
- types: List, Dictionary, Sorted list, Stack, Queue
